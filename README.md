# 阅读空间 – 本地原型

一个用于演示的本地网站，当前已具备注册/登录、书籍评论同步、每日任务赚金币以及照顾虚拟宠物等核心流程。

## 快速开始

1. **安装依赖**
   ```bash
   cd /Users/a8cde/Desktop/reading
   python3 -m venv .venv  # 可选，但推荐
   source .venv/bin/activate
   pip install -r requirements.txt
   ```

2. **初始化并运行项目**
   ```bash
   flask --app app run --debug  # 或 python app.py
   ```
   首次运行会自动创建/更新 `reading.db` 并写入示例书籍与任务。

3. **访问**
   打开浏览器访问 http://127.0.0.1:5000 。

## 主要功能
- **用户体系**：注册/登录/退出，密码以哈希方式保存在 SQLite。
- **书籍板块**：展示封面+书名，支持在书页下方发表评论。
- **年级书单**：预置六、七、八年级 24 本推荐图书，包含封面、作者、出版社信息。
- **讨论总板**：任意书页的评论会同步到全站讨论区，并附带返回书页的超链接。
- **每日任务与金币**：预置“读书打卡、写读后感、发布书评”三个任务；每天完成一次即可领取金币。
- **虚拟宠物**：金币可用于喂食、玩耍或送礼，实时更新宠物的饱食度和快乐值。

## 数据与结构
- SQLite 数据位于 `reading.db`，内含 `users / books / comments / tasks / task_completions / pets` 六张核心表。
- `books` 表保存 `author / publisher / grade / cover_url` 字段，可按年级分组展示；删除 `reading.db` 后重新运行可重新写入官方推荐书单。
- 每次启动会确保数据表存在，并自动补全 `books`、`tasks` 示例数据；如需重置可删除 `reading.db` 后重新运行。
